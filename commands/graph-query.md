## What the graph contains

- `stakeholders`: people and entities from `context/stakeholders.md`. Each has `id`, `name`, `category`, `fields` (bold-label key/value with a `placeholder` flag), and `line` (source line in the Markdown file).
- `relationships`: per-person entries from `memory/relationships.md`, same shape as stakeholders.
- `decisions`: dated entries from `memory/decisions.md`, with `date`, `decision`, `fields`, and `stakeholders` (resolved names).
- `priorities`: buckets keyed by slug (`this-quarter`, `this-month`, `this-week`, `on-hold`, `anti-priorities`). Each bucket has `items` (list) and `prose` (free text).
- `edges`: typed links between nodes.
  - `member_of`: stakeholder belongs to a category.
  - `last_interaction_at`: stakeholder has an interaction logged with a date.
  - `mentions`: decision references a stakeholder by name.
  - `mentions_unresolved`: decision mentions a name not in `stakeholders` (signal to backfill).
  - `no_logged_interaction`: stakeholder has no entry in `memory/relationships.md` (signal to backfill).
- `stats`: counts including `placeholder_count` and `stale_relationship_count`.

## Common queries

- "Who is in <category>?" → filter `stakeholders` by `category`.
- "Who have I not talked to in N days?" → walk `edges` of type `last_interaction_at`, filter by `when < today - N`. Also surface `no_logged_interaction` edges as relationships with zero log.
- "What decisions touched <person>?" → filter `decisions` for `stakeholders` containing the name, or follow `mentions` edges from `decision:*`.
- "What is on my plate this <quarter | month | week>?" → return `priorities[<bucket>].items` and `prose`.
- "Which stakeholder entries are placeholders?" → filter `stakeholders` for any `fields[k].placeholder === true`. Group by stakeholder so I see which entries need backfill.
- "Which decisions mention a name I do not track?" → list edges of type `mentions_unresolved`.

## Output

If the answer is affected by `_To fill in._` placeholders, flag that explicitly so I know which file to backfill.

If the graph is older than 7 days, note that you regenerated it before answering.
